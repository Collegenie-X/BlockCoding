## 📗 2주차: 러닝 게임 기획

### 🎮 템플릿 구조

```mermaid
sequenceDiagram
    participant 게임
    participant 캐릭터
    participant 장애물
    participant 점수
    
    게임->>캐릭터: 자동으로 달리기
    loop 매 프레임
        캐릭터->>게임: 스페이스 누르면 점프
        장애물->>게임: 왼쪽으로 이동
        게임->>게임: 충돌 검사
        alt 충돌 안함
            점수->>점수: +1점
        else 충돌함
            게임->>게임: 게임 오버
        end
    end
```

### 📝 1차시: 난이도 기획 (40분)

| 시간 | 활동 | 내용 | 교사 역할 |
|------|------|------|-----------|
| 5분 | 🎮 게임 플레이 | - 기본 템플릿 플레이<br>- 난이도 체감하기<br>- "너무 쉬워요? 어려워요?" | 데모 및 토론 |
| 10분 | 📊 난이도 분석 | - 무엇이 어렵게 만들까?<br>  * 속도? 장애물 개수? 간격?<br>- 워크시트 작성 | 사고 유도 |
| 20분 | 🎚️ 조정 실습 | - 장애물 속도 변경<br>- 간격 조절<br>- 테스트하며 조정 | 순회 지도 |
| 5분 | 🗣️ 발표 | - 내가 만든 난이도 설명<br>- 친구들이 플레이해보기 | 피드백 |

### 📋 난이도 설계 워크시트

```markdown

장애물 크기: _____ (작음/보통/큼)

🎁 아이템 설정:
□ 무적 아이템 (지속시간: ___초)
□ 점프력 UP (높이: ___)

💯 점수 설정:

아이템 먹으면: +___ 점

🎮 테스트 결과:
3번 플레이해서 평균 점수: _____

```

### 📝 2차시: 새로운 요소 추가 (40분)
