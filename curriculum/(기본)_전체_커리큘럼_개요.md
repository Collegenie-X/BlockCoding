# 🎮 게임 기획자로 성장하는 AI 블록코딩 커리큘럼

## 📌 핵심 교육 철학

### "만드는 사람이 아닌, 기획하는 사람으로!"

```mermaid
graph LR
    A[게임 기획자] --> B[아이디어 구상]
    B --> C[템플릿 선택]
    C --> D[나만의 기능 추가]
    D --> E[테스트 & 개선]
    E --> F[친구들과 공유]
    F --> B
    
    style A fill:#ff6b6b
    style C fill:#4ecdc4
    style D fill:#ffe66d
    style F fill:#95e1d3
```

### 🎯 왜 템플릿 기반인가?
- ✅ **빠른 성취감**: 5분 안에 작동하는 게임 체험
- ✅ **창의성 집중**: 코딩보다 아이디어와 기획에 집중
- ✅ **실패 두려움 제거**: 이미 작동하는 기반 제공
- ✅ **기획자 마인드**: "어떻게 더 재미있게 만들까?" 사고

## 🎨 템플릿 활용 3단계 학습법

```mermaid

        A1[선생님 템플릿 실행] --> A2[게임 플레이해보기]
        A2 --> A3[어떻게 작동하는지 관찰]
    end
    
    subgraph Step2[2단계: 기획하기]
        B1[뭘 바꾸면 재미있을까?] --> B2[기획서 작성]
        B2 --> B3[수정 계획 세우기]
    end
    
    subgraph Step3[3단계: 만들기]

    end
    
    Step1 --> Step2
    Step2 --> Step3
    Step3 --> D[친구들에게 공유]
    D --> E[피드백 받기]
```

### 📅 1주차: 캐릭터 게임 기획자 되기

```mermaid
gantt
    title 1주차 수업 흐름 (2차시)
    dateFormat mm
    section 1차시
    게임 체험(5분)           :01, 05m
    기획 아이디어(10분)       :06, 10m
    템플릿 수정(20분)         :16, 20m
    테스트(5분)              :36, 05m
    
    section 2차시
    지난 시간 복습(3분)       :01, 03m
    추가 기능 기획(7분)       :04, 07m
    자유 창작(25분)          :11, 25m
    발표 및 공유(5분)         :36, 05m
```

**🎯 학습 목표**
- 블록코딩 인터페이스 익히기
- 캐릭터 외형 변경하기
- 나만의 캐릭터 콘셉트 기획하기

**🎮 제공 템플릿: "마법의 드레스룸"**
- 기본 캐릭터 1개
- 옷 5가지 (모자, 상의, 하의, 신발, 액세서리)
- 색상 변경 블록
- 저장 기능

**📝 기획 미션**
1. 내 캐릭터 콘셉트 정하기 (예: 우주인, 공주, 닌자 등)
2. 추가하고 싶은 아이템 3가지 그리기
3. 특별한 애니메이션 1가지 기획하기

**⚡ 수정 포인트 (선택)**
- [ ] 옷 색상 바꾸기
- [ ] 새로운 액세서리 추가하기
- [ ] 배경 변경하기
- [ ] 캐릭터 표정 바꾸기
- [ ] 음악 추가하기

**🎨 기획자 워크시트**
